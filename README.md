# Jeu d'équilibre avec MPU6050

Projet réalisé sur STM32F746NG-DISCO.

## Objectif

Déplacer une bille sur l'écran en inclinant la carte grâce au capteur MPU6050.

## Matériel

- STM32F746NG-DISCO
- MPU6050
- Carte d'adaptation réalisée sous KiCad

## Fonctionnalités

- Lecture de l'accéléromètre
- Déplacement de la bille
- Zone de score mobile
- Chronomètre
- Menu de démarrage
- Victoire à 300 points
